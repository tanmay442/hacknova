# HackNova Quant Analytics Project

This repository contains an end-to-end equity analytics workflow for 15 NSE stocks + Nifty 50 benchmark (`^NSEI`) across 2023-2024.

It includes:
- data ingestion and cleaning
- risk/return analysis
- technical signal detection
- portfolio optimization
- stress testing
- a Streamlit dashboard for interactive viewing

## Objective

Build a reproducible analytics workflow to compare stock-level and portfolio-level risk-adjusted performance, then communicate insights through CSV outputs, charts, and a dashboard.

## Project Layout

- `fetch_data.py`: downloads raw OHLCV data from Yahoo Finance to `raw_data/`
- `datacleaning.py`: aligns stock data with Nifty trading dates and imputes small gaps into `cleaned_data/`
- `RiskReturnAnalysis.py`: Task 2 metrics + risk/return and correlation visuals
- `technicalsignladash.py`: Task 3 SMA crossover signal table + charts
- `portfolio_opt.py`: Task 4 portfolio design, comparison, and best/worst stock outputs
- `chaos.py`: Task 5 market crash stress test and safety ranking
- `stramalit_display/dashboard.py`: Streamlit UI to view all tasks interactively
- `outputs/`: generated artifacts from tasks

## Data Universe

- Banking: HDFCBANK.NS, ICICIBANK.NS, SBIN.NS, KOTAKBANK.NS, AXISBANK.NS
- IT: TCS.NS, INFY.NS, WIPRO.NS, HCLTECH.NS, TECHM.NS
- Pharma: SUNPHARMA.NS, DRREDDY.NS, CIPLA.NS, DIVISLAB.NS, APOLLOHOSP.NS
- Benchmark: ^NSEI

## Environment Setup

Use Python 3.10+ (recommended in a virtual environment).

Install required packages for scripts:

```bash
pip install pandas numpy scipy matplotlib seaborn yfinance
```

Install Streamlit UI dependencies:

```bash
pip install -r stramalit_display/requirements.txt
```

## Run Order (Recommended)

Run from repository root in this order:

```bash
python3 fetch_data.py
python3 datacleaning.py
python3 RiskReturnAnalysis.py
python3 technicalsignladash.py
python3 portfolio_opt.py
python3 chaos.py
```

## Streamlit Dashboard (View-Only)

The app is located at `stramalit_display/dashboard.py`.

Run it with:

```bash
cd stramalit_display
streamlit run dashboard.py
```

Important:
- This dashboard is for visualization and exploration only.
- It does not execute trading, place orders, or persist portfolio decisions.
- It is intended to present and explain analytics outputs interactively.

The sidebar includes:
- Risk Return Analysis
- Technical Signals
- Portfolio Optimization
- Chaos Stress Test

## Data Requirement for Streamlit

The dashboard reads CSVs from:
- `stramalit_display/data/cleaned_data/`

If this folder is missing or stale, copy latest cleaned files from project root:

```bash
mkdir -p stramalit_display/data
cp -r cleaned_data stramalit_display/data/
```

## Notes

- All annualization uses `TRADING_DAYS = 252`.
- Risk-free rate used in Sharpe calculations is `6.5%`.
- The optimizer can fail if no stock passes the Golden Cross filter; this is guarded in Task 4.
- Task 4 plotting uses non-interactive backend (`Agg`) for terminal/headless stability.
